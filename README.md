# Usage:

```
"custom/calendar": {
    "exec": "~/.local/share/waybar/waybar-simple-calendar/calendar.sh",
    "interval": 3600,
    "return-type": "json",
    "on-click": "~/.local/share/waybar/waybar-simple-calendar/calendar.sh",
    "signal": 8
}
```

Make it executable:

```
chmod +x calendar.sh
```
